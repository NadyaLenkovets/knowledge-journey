# Knowledge Journey

Интерактивная система обучения: тема или текст → AI генерирует journey → прохождение под таймером → финальный отчёт.

Стек и визуал основаны на [Prompt Lab](https://github.com/NadyaLenkovets/prompt-lab) (Модуль 1).

## Статус

Проект в разработке. Прогресс по этапам: [PROGRESS.md](./PROGRESS.md).

## Локальный запуск (после Этапа 0+)

```bash
npm install
npm run dev
```

Подробности появятся по мере готовности этапов (Hono API, `.env` с OpenRouter — на Этапе 4).
